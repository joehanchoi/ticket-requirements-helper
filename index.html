Trello Requirements Helper Tool

Helps convert tabular field data to be ready for Trello tickets.

Tool

Instructions

  1. Your tabular data should not have any line breaks and should be in the following format:
    2. Field Field Type Required? Helper Text
    Page Title Plain Text, Short Required The title for the page.
    Page Description Plain Text, Long Not Required The description for the page.
  2. Copy all your table data except the table headings straight from the table.
  3. Paste this text string into the Input textbox.
  4. Press the Get Trello Text button. The formatted text will be outputted below.
